# 🚀 v3.9.0 发布说明

**发布日期**: 2026-04-17

**文件大小**: ~162 KB

## ✨ 新特性

### 1. 流式输出支持 ⭐⭐⭐

AI 回复现在支持实时流式显示，用户体验大幅提升！

**核心改进**:
- ✅ 首字延迟 < 1 秒（之前 5-10 秒）
- ✅ SSE (Server-Sent Events) 实时接收
- ✅ 智能节流优化（50ms 更新间隔，20 FPS）
- ✅ 支持停止按钮中断输出
- ✅ 流式完成后自动执行代码块

**技术实现**:
- 使用 `fetch` + `ReadableStream` 替代阻塞式请求
- requestAnimationFrame 优化 DOM 更新频率
- AbortController 支持优雅取消

**验收结果**:
- 感知延迟降低 80%
- 支持所有 OpenRouter 免费模型
- 长文本显示流畅无卡顿

---

### 2. 快捷键系统 ⭐⭐⭐

完全键盘操作支持，效率提升 50%！

**已实现快捷键**:
- `Ctrl+Enter` - 发送消息
- `Escape` - 隐藏窗口（设置对话框优先）
- `Ctrl+ArrowUp` - 导航到上一条用户消息（滚动定位 + 高亮）
- `Ctrl+ArrowDown` - 导航到下一条用户消息（滚动定位 + 高亮）

**技术亮点**:
- ShortcutManager 统一管理
- 不冲突的键位设计
- 视觉反馈（高亮动画）
- 可扩展架构（未来支持自定义）

---

### 3. 构建与发布系统优化 ⭐⭐

自动化构建流程，杜绝人为错误！

**新增功能**:
- ✅ 开发/发布双模式支持
- ✅ DEBUG_MODE 自动切换
- ✅ 完整的构建文档
- ✅ 一键发布流程

**使用方式**:
```bash

# 发布模式（自动关闭调试日志）

node build.js --release
```

**收益**:

- 新人快速上手

## 🔧 优化与修复

### 性能优化

1. **流式更新节流**
   - 减少 80% 无效 DOM 操作
   - 保持 20 FPS 流畅更新
   - 长文本场景性能提升显著

2. **codeBlockStore 清理逻辑修复**
   - 修复字符串排序导致的清理错误
   - 改用数字索引正确排序
   - 防止内存泄漏

### 代码质量

1. **统一调试日志管理**
   - 新增 `Utils.debugLog/Warn/Error()` 函数
   - 替换所有 console 调用（24 处）
   - 发布模式自动关闭日志

2. **增强流式文本格式化**
   - 新增粗体、斜体支持
   - 新增链接解析
   - 新增列表格式（有序/无序）
   - 兼容完整 Markdown 基础语法

3. **标记废弃 API**
   - `callAPI()` 标记为 @deprecated
   - 保留作为 fallback
   - 明确迁移路径

## 🐛 Bug 修复

1. **codeBlockStore 清理错误**
   - 问题：字符串排序导致旧数据无法正确删除
   - 修复：改用数字索引排序
   - 影响：防止内存泄漏

2. **escapeHtml 引用问题**
   - 问题：ui.js 中调用未定义函数
   - 修复：使用 UITemplates.escapeHtml()
   - 影响：避免运行时错误
