# Free Web AI Agent

基于 AI 模型的浏览器 AI 助手，支持 JavaScript 代码执行。

## ✨ 核心功能

- 💬 **智能对话** - 支持多种 AI 模型
- 💻 **代码执行** - 直接在浏览器中执行 JavaScript
- 🆓 **免费使用** - 支持 OpenRouter 免费模型

## 🚀 快速开始

### 安装

1. 安装 Tampermonkey 浏览器扩展
2. 获取 API Key (OpenRouter / DashScope)
3. 下载 [`dist/agent.user.js`](dist/agent.user.js)
4. 拖拽到浏览器窗口完成安装

### 配置

1. 刷新网页，点击右下角 "✨ Free Web AI Agent" 按钮
2. 点击 "⚙️ 设置" 粘贴 API Key
3. 选择模型，保存配置

## 📦 构建

```bash
node build.js
# 输出: dist/agent.user.js
```
